# refs

Optional reference photos for the self grading loop. This folder can be empty.

If you drop image files named for a species here, the nightly grader will open them
as visual references and compare its renders against them. Suggested names:

- `sunflower.jpg` (or .png)
- `blue_rose.jpg`
- `spider_lily.jpg`

Use images you have the right to use. The grader works without these too, it falls
back to the rubric and known botanical features when no reference image is present.
