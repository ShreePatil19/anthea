# Credits and design references

anthea draws its procedural flower techniques from the following open projects,
which served as design references and inspiration. The code in this repository
is an original implementation; no source files were copied verbatim.

## saud-26 / Garden-in-Bloom

Procedural OpenCV flowers with petal layering and disc geometry.
https://github.com/saud-26/Garden-in-Bloom

## barmoshe / bloom-garden

Bloom animation and colour palette approaches for stylised flowers in OpenCV.
https://github.com/barmoshe/bloom-garden

## misalavinash / flower-generator

Generative petal geometry and parametric flower rendering.
https://github.com/misalavinash/flower-generator

---

anthea extends these references with real golden angle seed packing (Fibonacci
spirals), anatomically correct spider lily stamens, supersampled anti-aliasing,
and adaptive HSV gradient palettes.
